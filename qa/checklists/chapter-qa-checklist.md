# Chapter QA Checklist

ใช้ checklist นี้หลังแปลทุกตอน

## Completeness

- [ ] แปลครบทุกย่อหน้า
- [ ] ไม่ข้ามบทสนทนา
- [ ] ชื่อบทถูกต้องและล้าง markup แล้ว

## Accuracy

- [ ] ความหมายตรงต้นฉบับ
- [ ] ไม่มีการสลับพี่/น้อง
- [ ] ไม่มีการสลับพ่อ/ลูก อาจารย์/ศิษย์
- [ ] เหตุการณ์และลำดับเวลาถูกต้อง

## OKF Consistency

- [ ] ชื่อคนตรง OKF
- [ ] ชื่อสำนัก/ฝ่ายตรง OKF
- [ ] ชื่อวิชาตรง OKF
- [ ] ชื่อวัตถุ/สมบัติตรง OKF
- [ ] คำเรียกตำแหน่งตรง OKF

## Thai Fluency

- [ ] อ่านเป็นนวนิยายไทย
- [ ] บทสนทนาเป็นธรรมชาติ
- [ ] น้ำเสียงตัวละครแยกกันได้
- [ ] ไม่มีประโยคแข็งแบบแปลตรง

## Clean Output

- [ ] ไม่มีเศษอังกฤษ
- [ ] ไม่มีวงเล็บกำกับศัพท์อังกฤษในเนื้อเรื่อง เช่น `'สามผนึก' (Three Seals)` (เว้นแต่ต้นฉบับมีจริง)
- [ ] ไม่มีเศษเกาหลี
- [ ] ไม่มีเศษจีน
- [ ] ไม่มี markup
- [ ] ไม่มี note ผู้แปลหลุด
- [ ] ไฟล์เปิดด้วย UTF-8 อ่านเป็นไทยได้จริง ไม่มี mojibake (เธ/เน€ เพี้ยน) — รัน `etc/check-encoding.ps1` ได้
