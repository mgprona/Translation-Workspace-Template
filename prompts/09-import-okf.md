# Prompt: Import OKF จากงานเก่า

ใช้เมื่อ **แปลต่อจากโปรเจกต์ที่เคยแปลแล้วบางส่วน** (มี OKF เดิม หรือมีบทแปลไทยที่ล็อกศัพท์ไว้แล้ว) — เพื่อให้คำแปลใหม่ตรงกับมาตรฐานเดิม **แทนการ bootstrap ใหม่**

> เหตุผล: ถ้า bootstrap ใหม่จาก source เปล่า AI จะเลือกคำตามดุลพินิจตัวเอง ซึ่งมักไม่ตรงกับที่เคยแปลไว้ (เช่น "เทพมาร" vs "จอมมาร" — ถูกทั้งคู่แต่คนละคำ) ทำให้ทั้งเรื่องคำแปลแกว่ง

## เมื่อไรใช้ prompt นี้แทน 08-bootstrap

- มี OKF เดิมจากโปรเจกต์ก่อน (ไฟล์ `okf/*.md` ที่มีข้อมูลแล้ว) → **ใช้ prompt นี้**
- มีบทแปลไทยเก่า (`thai_final/` หรือไฟล์แปล) แต่ OKF ไม่ครบ → ใช้ prompt นี้ดึงศัพท์จากบทแปล
- เริ่มเรื่องใหม่ ไม่มีอะไรมาก่อน → ใช้ `08-bootstrap-okf.md` ตามปกติ

## ขั้นตอน

### 1. ระบุแหล่ง OKF เดิม

ถามผู้ใช้ว่า OKF/บทแปลเดิมอยู่ที่ใด (path โฟลเดอร์โปรเจกต์เก่า)

### 2. นำเข้า OKF โดยตรง (ถ้ามีไฟล์ okf เดิม)

- คัดลอกเนื้อหาตาราง OKF เดิม (characters, terms, factions, places, techniques, artifacts, title-registry, voice-register) มาลงไฟล์ OKF ของโปรเจกต์นี้
- **ห้ามเปลี่ยนคำแปลเดิม** — ยึดเป็นมาตรฐานถาวร (one term, one form)
- ถ้าโครงสร้างคอลัมน์ต่างกัน ให้ map เข้าคอลัมน์ที่ตรงกัน อย่าทิ้งข้อมูล

### 3. สกัดศัพท์จากบทแปลเดิม (ถ้าไม่มีไฟล์ okf แต่มีบทแปล)

- อ่านบทแปลไทยเก่า 3-5 ตอน เทียบกับ source อังกฤษตอนเดียวกัน
- สกัดว่าศัพท์เฉพาะแต่ละคำถูกแปลเป็นไทยว่าอะไร แล้วลง OKF
- ตรวจความสม่ำเสมอ — ถ้าบทแปลเก่าเองก็แกว่ง ให้เลือกรูปที่ใช้บ่อยสุด แล้ว mark คำที่แกว่งใน `human-review-needed.md`

### 4. ตรวจ genre profile

ตั้ง genre profile ใน `style-guide.md` ให้ตรงแนวเดิม (ดูจากสำนวนบทแปลเก่า)

### 5. แทน placeholder + verify

แทน `{DATE}`/`{NOVEL_NAME}` แล้วรัน `etc/check-placeholders.ps1` (ดู SETUP.md)

### 6. สรุปให้ผู้ใช้

- จำนวนศัพท์ที่ import เข้าแต่ละไฟล์ OKF
- คำที่บทแปลเก่าแกว่ง (อยู่ใน `human-review-needed.md`)
- ยืนยันว่าพร้อมแปลต่อโดยคำจะตรงกับงานเดิม

## ข้อห้าม

- ห้าม "ปรับปรุง" คำแปลเดิมให้ดีขึ้นโดยพลการ — แม้คำเดิมจะไม่สวย ก็ต้องคงไว้เพื่อความต่อเนื่อง (ถ้าอยากเปลี่ยนจริง ให้ลง `human-review-needed.md` status `replace` พร้อมเหตุผล)
- ห้าม bootstrap ใหม่ทับ OKF เดิม
- ห้ามเดาคำที่บทแปลเก่าไม่เคยกำหนด แล้วล็อกเป็นมาตรฐาน — ใส่ `human-review-needed.md`
